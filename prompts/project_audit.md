# Prompt — Project Audit

Use this when checking whether the repo is complete and reproducible.

```text
Audit my NBA Fantasy Decision System project.

Review whether the project has:
1. README
2. requirements.txt
3. .gitignore
4. src/nba_fantasy modules
5. scripts
6. tests
7. references documentation
8. prompts markdown files
9. data/raw, data/interim, data/processed, data/league_snapshots, data/outputs, data/run_manifests folders
10. sample projection files
11. Flaim-style snapshot files
12. dated snapshots
13. dated reports
14. run manifests
15. assumptions log
16. data dictionary

Identify:
- Completed components
- Missing files
- Missing prompts
- Missing data inputs
- Missing tests
- Next highest-value phase

Be blunt and separate critical blockers from nice-to-have improvements.
```
