# NBA Fantasy Decision System

A Python + connector-assisted decision system for Yahoo NBA 9-category fantasy basketball.

## Purpose

This project helps make data-informed decisions for:

1. Drafting players
2. Start/sit decisions
3. Free-agent adds
4. Drop decisions
5. Trade proposals and trade evaluation

## Core tools

- Python for data cleaning, scoring, rankings, and repeatable analysis
- Flaim Fantasy connector for Yahoo league context
- DraftKings connector only if useful for NBA schedule or game context
